Git est un programme permettant de gérer les différentes versions d'un projet (fichier)

Git permet de sauvegarder chaque changement dans le fichier ou projet (versionning), accompagné par un commentaire (commit) sur le changement

Voici les principales commandes

git init Crée un dépôt local à partir du nom spécifié ( il faut être dans le répertoire )
git status ( et pas git statut !!!) Liste tous les nouveaux fichiers et les fichiers modifiés à commiter en cas d'erreur un message indique la marche à suivre
git add nomfichier Ajoute un instantané du fichier, en préparation pour le suivi de version
$ git config --global user.name "[nom]"
